Комманда Описание
init Инициализация папки .git
status Показывает текущию ветку, информацию о коммитах, untracked files
add ->file<- Отслеживать этот файл либо зафиксировать изменение файла перед коммитом
add . Отслеживать все файлы
commit -m "name commit"
branch Тек ветка
git branch ->name<- Новая ветка
git branch -D ->name<- Удалить ветку
git checkout ->branch name<- Переключение между ветками
git branch -b ->branch name<- создать и переключиться на создданную ветку
git merge 'название ветки' складывает две ветки
git push -u origin master Залить на сервер
git remote add origin git@github.com:Grayzone90/git-exploy.git Удалеёное соединение
git clone ссылка с репозитория клонирование проекта
